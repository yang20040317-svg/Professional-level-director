*   **通用中文提示词模板**：

*   **美学阐述**：将中国传统绘画的“留白”、“气韵”与现代光影透视相结合。写意注重墨色五彩（焦、浓、重、淡、清）的流动与宣纸的晕染质感；工笔重彩注重细腻的线条勾勒与矿物颜料（石青、石绿、朱砂）的层层叠加。
