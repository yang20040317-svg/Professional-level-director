# 画面逆向反推与灵魂重构心法（reverse_engineering_soul.md）

> [!IMPORTANT]
> **核心痛点**：普通 AI 反推（Describe）只无序堆砌画面中的元素名称，却**不提取画面生成的物理与构图逻辑**，导致复刻出来的提示词再次跑图时像在“抽卡”，根本复刻不出原图的灵魂与美学气韵。
> **逆向核心哲学**：风格是皮，结构是骨。只有逆向提取出参考图的**「结构骨架」**与**「核心不变项」**，才能实现 100% 的中文意境复刻。
> **万能还原公式**：
> $$\text{完美中文还原提示词} = \text{核心不变项（构图/色彩/氛围）} + \text{结构骨架（机位/光影/空间）} + \text{细节材质修饰} + \text{限制性降噪句式}$$

## 一、 中文灵魂反推的三大核心方法 (Three Core Methods)

对参考图进行逆向拉片时，严禁平铺直叙地翻译，必须遵循以下三大步骤用中文提取画面灵魂：

### 1. 锁定不变项（Locking the Invariants）
*   **做法**：首先向创作者明确，原图中**最想保留的核心美学特质**是什么（是极窄景深下的微距情绪，还是清冷幽深的林间色彩，亦或是大面积留白的静谧氛围？）。
*   **控制手段**：在组装中文提示词时，将所有描述权重和墨水高度集中在这个核心特质上，强行过滤掉画面中无关紧要的噪杂细碎元素，实现意境优先。

### 2. 生成结构骨架（Generating Structural Skeleton）
*   结构骨架决定了新图的空间立体度与透视。必须从以下三个物理维度进行专业中文术语锁定：
    *   **构图机位 (Camera Setup)**：极低机位地面仰拍、85mm人像定焦、极浅景深近景特写、过肩对角线对峙景别。
    *   **光影物理 (Lighting Physics)**：戏剧性高反差单向侧光源（伦勃朗三角光）、穿透空气的耶稣体积光束（丁达尔光路）、大体积漫反射瑞利散射。
    *   **空间关系 (Spatial Relations)**：前景黑色剪影树枝折挡（提供透视深度）、中景人物主体、背景无限深远消融消逝的地平线。
*   **术语锁死**：用这些确凿的镜头物理词锁定画面，AI 绝对无法在下一轮中随机乱出。

### 3. 验证纠错闭环（Verification & Error Loop）
*   **工作流**：`原图解析 -> 生成中文提示词 -> 跑图生成新图 -> 对比原图与新图 -> 二次修正中文词`。
*   **对比纠错**：引导 AI 充当纠错裁判，主动对比原图与新图在光线、透视上的偏差：
    *   *“原图中的金边发丝逆光在新图中丢失了”* $\to$ 二次修正：在正面提示词中追加“阳光强烈逆光照射，在女孩发丝边缘镀上一道极其耀眼的金边”。
    *   *“新图的深邃感不够”* $\to$ 二次修正：在段尾加重限制句式：“暗部深邃温润，阴影过渡极其柔和，无无故发光与溢光”。

```
       [ 完美还原公式组装链条 ]
1. 核心不变项：神秘幽暗的苔藓暗绿原始森林氛围，强反差丁达尔光束与麋鹿。
2. 结构骨架：15mm超广角极低机位仰视，单向顶部强体积光，前景树叶剪影折挡。
3. 细节修饰：湿润水雾，麋鹿皮毛水珠反光，哈苏相机质感。
4. 降噪句式：画面清澈无烟雾噪点，拒绝塑料胶感，光影符合真实物理无溢光。
```
