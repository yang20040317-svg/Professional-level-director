# AI 故事板视频稳定性控制工作流与核心模板集 (storyboard_stability_workflow.md)

> [!IMPORTANT]
> **行业终极痛点**：纯靠提示词生成视频，极易导致镜头无逻辑（AI 不懂景别与运镜）、人物变脸换装、以及风格色调前后混乱。
> **核心解决方案**：**AI 视频的不稳定，是因为没有提前理解拍摄逻辑。** 引入「AI 故事板工作流」，在人物身份、剧情、镜头线稿和风格样片四个物理维度提前建立极限制约，让最终生成的视频全程如铁桶般稳定。

---

## 一、 稳定生成 AI 视频的标准工业流程

```
 ┌─────────────────┐       ┌─────────────────┐       ┌─────────────────┐
 │  1. 确定人物IP  │ ───>  │  2. 文本分镜剧情 │ ───>  │  3. 生成故事板   │
 │   (三视图固定)   │       │   (15秒6-8镜)   │       │  (镜头规划/站位)  │
 └─────────────────┘       └─────────────────┘       └─────────────────┘
                                                              │
                                                              ▼
 ┌─────────────────┐       ┌─────────────────┐       ┌─────────────────┐
 │ 最终一键生成视频  │ <───  │ 5. 情绪口型配音  │ <───  │  4. 预想效果图   │
 │   (多图融合I2V)  │       │   (作为情绪参考) │       │  (色调风格样片)  │
 └─────────────────┘       └─────────────────┘       └─────────────────┘
```

1.  **确定人物（人物身份板）**：先做男女主定妆图与人物三视图，锁死面部五官、特定发型和服装细节。
2.  **生成剧情**：基于固定好的人物，生成 6-8 镜、15 秒左右、完全画面化的物理动作分镜剧本。
3.  **生成 AI 故事板**：生成铅笔素描风格的故事板格格分镜，标注景别、主要动作、运镜与人物构图位置（故事板只负责镜头规划，不负责最终画质）。
4.  **生成预想效果图**：统一整体视频的光影、冷暖色温、体积阴影与电影感画面质感，作为最终视频风格的“Style Reference（sref 风格样片）”。
5.  **补充对话情节**：按分镜配自然对话，仅作为情绪和口型渲染参考，画面不生成字幕。
6.  **最终生成视频**：整合人物 IP、故事板构图、效果图色调，用极简的“物理连锁动作词”执行图生视频（I2V）或文生视频（T2V）。

## 二、 避坑指南：使用 AI 故事板的 3 个常见错误

*   **❌ 错误 1：无支撑直接生成故事板**
    *   *后果*：一开始就直接生成故事板，导致角色面部五官和装束随机漂移，后续完全无法拼合。必须先确立“人物IP定妆板”和“文字剧情分镜”作为底层骨架。
*   **❌ 错误 2：混淆镜头规划与人物细节**
    *   *后果*：在故事板线稿阶段贪多，写入了过于复杂的画质、颜色和微小服装花纹描述，导致 AI 故事板画面失控。记住：**故事板只负责镜头构图、纵深透视规划与人物站位！**
*   **❌ 错误 3：提示词冲突与自我推翻**
    *   *后果*：最终生成视频时，提示词与前期设定的人物三视图、效果图的描述词产生刚性冲突，导致画面崩坏。最终提示词应极简，**只负责描述“当前的物理力学动作与环境连锁互动”**。

## 三、 五套核心提示词模板 (可直接复制)

### 📋 模板 1：人物三视图提示词 (锁死IP形象)
*   **使用平台**：Midjourney / Stable Diffusion / Flux (用于奠定正面与多视角细节)
*   **模板公式**：`[主体身份] + 角色三视图（正面立像，侧面立像，背面立像）+ 统一淡灰色studio中性背景 + 极其细致的服装材质发型描述 + 限制性降噪句式组`
*   **👉 可直接复制模板**：
    > 一幅高规格的商业角色概念设计三视图，展现一位高贵的唐代女刺客。画面包含三个标准视角（标准正面立像，九十度侧面立像，标准背面立像），置于统一干净的浅灰色studio背景中。角色面部五官一致（凤眼微挑，神情冷峻），发饰发型完全固定一致（高耸云髻，黑发如墨），服装细节一致（贴身玄色暗纹织锦刺客劲装，肩部有精细的暗金色暗夜云纹金属铠甲，腰间系有紧绷的黑色皮革束腰，小腿绑有匕首护鞘）。强环境遮挡（AO）阴影，无杂乱背景干扰。画面干净清澈，无烟雾噪点，皮肤呈现哑光真实肌理，拒绝变脸与换装。

### 📋 模板 2：剧情生成提示词 (ChatGPT/Claude分镜创作)
*   **使用平台**：ChatGPT / Claude / Kimi (用作文字大纲和分镜脚本生成)
*   **👉 可直接复制模板**：
    > 你是一位拥有顶级运镜直觉的电影分镜导演。请基于角色“唐代女刺客”和“雨夜刺杀”的主题，为我编写一段15秒内、包含6个核心镜头的画面化简单剧情分镜脚本。
    > 请严格按照以下格式输出每一个镜头：
    > 【第N镜 : 景别与机位】-> 画面描述（强调物理动作、重力对抗与环境连锁物理交互，如“鞋底溅起水滴”、“风吹起斗笠”）-> 运镜方式（如“低机位快速跟摇”、“机位平滑惯性前推”）。
    > 注意：请全面废除空洞的质量形容词（如“超清”、“史诗级”），每个镜头的画面描述必须是摄影机可以捕捉到的客观物理瞬间。

### 📋 模板 3：故事板生成提示词 (镜头与构图规划)
*   **使用平台**：Midjourney (使用 `--ar 16:9` 格式) / Stable Diffusion
*   **模板公式**：`电影手绘故事板线稿 + 极简黑白灰度调 + 快速排线笔触 + 景别与构图描述 + 人物站位 + 机位运动指示箭头`
*   **👉 可直接复制模板**：
    > 一幅极具专业电影感的手绘故事板分镜线稿。采用干净的手绘铅笔素描线稿风格，极简黑白灰度调，带有艺术性的快速排线笔触。画面清晰标注出景别与机位运动指示箭头。画面内容：一名戴斗笠的剑客偏置在画面右侧三分线，正处于拔剑格挡的蓄力瞬间，长剑在空中留下一条带有速度感的指示弧线。只负责镜头构图、纵深透视规划与人物站位，无复杂色彩。画面干净清澈，排线工整，具有强烈的动态指示感。

### 📋 模板 4：预想效果图生成提示词 (确定光影色调风格)
*   **使用平台**：Midjourney (作为 `sref` 使用) / SD / Flux (作为奠定色调和氛围的基底)
*   **模板公式**：`[核心主体] + 单一强光源（月光/烛光/霓虹）+ 高反差明暗对比（Chiaroscuro）+ 电影胶片冷暖色温 + 环境折射介质（雨水/体积雾/灰尘微粒）`
*   **👉 可直接复制模板**：
    > 一幅史诗级电影级的商业预想效果图，奠定整体画面的艺术基调。描绘唐代暗夜小巷中，单一高亮度青蓝色侧向月光照射在潮湿古旧的石板路面上，溅起淡淡的雨水雾气。整体色温偏冷偏深（5500K冷调），阴影深邃且充满体量感，光影高反差。空气中带有细小的微尘颗粒在光束中浮动。质感极高，带有轻微的电影胶片颗粒。画面干净清澈，无烟雾噪点与非物理乱光，色彩沉稳不刺眼，奠定冷峻幽深的叙事氛围。

### 📋 模板 5：最终视频生成提示词 (多图融合I2V/T2V)
*   **使用平台**：即梦 Seedance 2.0 / 可灵 / Runway Gen-3
*   **模板公式**：`[锁定首帧效果图基调与女刺客三视图IP] + 物理动作力学形变 + 环境连锁物理交互 + 机位运动 + 限制性降噪句式组`
*   **👉 可直接复制模板**：
    > 锁定首帧效果图冷峻基调与唐代女刺客玄色铠甲装束IP。极窄景深，低机位快速前推镜头。女刺客在倾盆大雨中迅猛转体拔出腰间短匕，匕首在空中划出一道亮白的反光弧线，雨滴重击在匕首刀身呈环状飽滿飞溅，风雨猛烈拉扯着她玄色风衣的衣摆向后狂舞，地面积水被她鞋底踩出层层扩散的同心圆涟漪，动作具有强烈的物理惯性与空气流速对抗。画面整体干净无杂质，无任何画外文字，动作连贯自然。
