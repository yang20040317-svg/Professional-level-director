# 中文废片重塑与多轮资产化心法（waste_remolding_assets.md）

> [!IMPORTANT]
> **核心哲学**：**90% 的废片不是垃圾，而是未开采的数字资产。**
> AI 生图是一个**多轮重构、不断逼近收敛的工程**，而不是靠运气的一键抽卡。通过将“废片”进行解构、锁定光影构图，或进行局部精准的“中文蒙版描述重绘”，可以实现画面质量 100% 的稳定收敛交付。

---

## 一、 中文三大废图资产化重塑维度 (Three Chinese Remolding Pillars)

当得到一张细节崩坏（如手部变异、背景多出杂物）但整体气韵出色的“废图”时，切勿直接删除。应将其拆解为以下三种高价值的中文资产：

### 1. 拆分复用：保留光影与构图（Locking Invariants）
*   **痛点**：画面的色彩调子、光影逻辑、空间透视极其惊艳，但是主体人物的脸部、手部崩了。
*   **中文做法**：把这张废图作为**“风格与构图参考图”**垫给生图平台。在生图指令中选择“参考图像”并锁定其“光影风格”与“骨架结构”。提示词只改主体名词，不重写光影，直接复刻底图精髓。
*   **技术指引**：在即梦等中文平台的“混图/垫图”模式下，上传废图并把强度滑块拉至 `0.5 - 0.7`（参考结构与氛围），正面提示词中直接将崩坏的主体替换为新的中文描写（如：“一名戴着细银丝边眼镜的严肃男学者，正在翻阅古籍”），实现光影的 100% 稳定保留。

### 2. 局部重绘：蒙版与精准中文描述（Chinese Inpainting）
*   **痛点**：画面的 99% 都非常完美，仅仅有 1% 的局部细节出错（例如：六根手指、鞋子样式不对、背景里多了一个杂乱路人）。
*   **中文做法**：使用 **Inpainting (局部重绘 / 局部修改)** 功能。用画笔涂抹崩坏的 1% 区域，**只写针对该区域的局部修正中文描述**，绝对不要动整体画面，实现无缝融合。
*   **高拟真局部重绘中文指令集**（涂抹后直接喂给 AI 的局部提示词）：
    *   *修正手部*：`“精细的右手，指关节微动握紧，手指不多不少正好五根，指甲边缘圆润，皮肤呈现毛孔可见的自然哑光肌理，无任何多余手指与关节畸变。”`
    *   *替换衣服*：`“素雅整洁的白色短袖棉麻连衣裙，面料质感柔软贴身，带有一丝自然的垂坠折痕，无多余反光塑料胶感。”`
    *   *清除路人/杂物*：`“纯净干净的灰色斑驳水泥墙面，带有自然的微弱灰尘斑驳与漫反射光影，没有任何杂物与路人遮挡。”`

### 3. 负面约束：提取中文“错题本”（Error Log）
*   **痛点**：AI 在多轮渲染中，反复顽固地在画面中生成某些刺眼的杂点、虚假光斑或脏色块。
*   **中文做法**：将这些错误特征精准提取，作为**限制与截断条件**加入正面提示词段尾，强行封锁 AI 的采样空间。
*   *错题截断范例*：
    > `“……（正面写完）。画面整体极其清澈，没有任何莫名生成的发光小球与漂浮光斑，无雾气噪点与虚化杂质，画面边缘光影完全符合物理，无溢光与假边缘线。”`

## 二、 通用中文多轮收敛工作流

在面对画面的瑕疵时，引导用户按下图所示的中文工作流进行定向微调纠偏：

```
                 [ 拿到初始生成图 ]
                         │
                   (发现局部瑕疵)
                         │
                         ▼
        [ 诊断：是光影好，还是局部有伤？ ]
           /                           \
          /                             \
    (整体光影极佳)                (99%完美, 1%硬伤)
        /                                 \
       ▼                                   ▼
  [ 1. 垫图锁定 ]                    [ 2. 局部重绘 ]
 (拖入混图/锁定光影)               (画笔涂抹崩坏区域)
         │                                 │
         ▼                                 ▼
[ 仅改主体正向词 ]                 [ 输入局部中文指令 ]
(如: 把少女换成少年)               (如: 完美的五指手部)
         └────────────────┬────────────────┘
                          │
                          ▼
                    [ 跑图迭代 ]
```

通过这套中文多轮迭代工作流，可以彻底告别“碰运气一键抽卡”的原始打法，像在数字雕刻室中一样，2-3轮即可稳健收敛交付高水准的商业级画卷。
