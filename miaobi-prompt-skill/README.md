# ✨ 妙笔生图描述技能 (Miaobi Prompt Skill)

> **妙笔生花，画境跃然纸上。** 基于纯中文自然语言的殿堂级 AI 画面描述与通用提示词架构系统。

---

## 🎨 项目简介

**「妙笔（Miaobi Prompt Skill）」** 是一款专为 **Midjourney、Stable Diffusion、Flux、DALL-E 3** 等主流 AI 生图平台打造的开源画面描述与提示词工程心法系统。

本系统全面废除了复杂的英文词汇拼凑与刻板的打标堆砌，**统一使用符合现代大模型语言直觉、极具画境张力的“纯中文自然语言描述流”**。它能够将您脑海中朦胧、粗糙的感性感受，经过物理学与艺术美学的精密重构，锻造成高保真、极富故事情感、光影真实的电影级画面描述。

---

## 🚀 核心设计哲学 (加减乘除逆)

妙笔的核心竞争力源于将物理学连锁反应、视觉心理学与艺术规律高度解耦的**「五大中文描述法则」**：

| 法则 | 核心心法 | 对应分册 |
| :--- | :--- | :--- |
| **➕ 加法** | **物理交互对抗 & 身份妆系统**。拒绝静态悬浮，以角色妆造定义灵魂。 | [👉 角色妆造美学](references/character_styling_aesthetics.md) |
| **➖ 减法** | **限制性“剪词降噪”**。用物理约束在正面描述中斩断 AI 的塑料假光与噪点。 | [👉 剪词降噪美学](references/jianci_noise_reduction.md) |
| **✖️ 乘法** | **废片重塑与多轮资产化**。AI 是多轮对话，将废片转化为可再利用的数字资产。 | [👉 多轮资产重塑](references/waste_remolding_assets.md) |
| **➗ 除法** | **感性具象与物理推演**。扮演中文视觉翻译官，在画面中反向推演生理微动。 | [👉 环境叙事心法](references/visual_translator_environmental.md) |
| **🔄 逆** | **逆向拉片与灵魂反推**。分析骨架三维，套用公式完美复刻任意参考图。 | [👉 画面逆向反推](references/reverse_engineering_soul.md) |
