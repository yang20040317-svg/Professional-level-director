## 🎨 项目简介

本项目是首个将**正向创意设计**与**逆向物理拉片**融为一体的 AI 生图与视频生成中文提示词心法系统。全面废除英文堆砌，统一使用符合大模型直觉、极具视觉张力的“纯中文自然语言描述流”，彻底解决 AI 生图的“塑料假光”与 AI 视频的“衔接跳跃、动作悬浮”等痛点。

📦 **两大核心核能板块**：
1. **即梦 Seedance 2.0 视频生成器**：专注视频动态力学、肢体物理对抗、多视角及镜头惯性平滑控制。
2. **妙笔 (Miaobi) 中文生图描述系统**：基于加减乘除逆五大法则，深度勾勒角色高级妆造、限制性降噪及眼神生理微动。

项目配备 10 余篇殿堂级物理与艺术心法分册，完美打通 Midjourney、Stable Diffusion、Flux、即梦等多平台适配。

## 🎞️ 第一板块：即梦 Seedance 2.0 视频提示词生成器 (详细介绍)

本板块专为字节跳动即梦平台的 **Seedance 2.0 视频生成模型** 编写高质量中文提示词，消除视频运动中的呆板悬浮，实现镜头与物理规律的行云流水。

### 1. 平台规格参数
*   **视频比例**：默认为 `16:9` 宽画幅电影景别。
*   **主体规格**：要求主体处于运动状态（如：奔跑、侧身跃起、回眸）。
*   **防爆与禁忌**：**绝对禁止**使用任何无意义的抽象形容词（如“史诗级画面”、“超清画质”），要求用纯物理反应、尘土飞扬等客观事实来替代质量词。

### 2. 连续分镜惯性推演公式
为了消除前后两帧视频“跳帧、不连贯”的致命痛点，本系统引入**「惯性推演公式」**：
$$\text{下一分镜提示词} = \text{[锁定前帧基调]} + \text{[动作力学形变]} + \text{[机位随动位移]} + \text{[环境连锁反应]} + \text{[降噪限制句式]}$$
*   **前帧基调**：锁定角色服装材质、色调、光源位置。
*   **力学形变**：物理位移产生的重力对抗（如“鞋底与积水猛烈撞击飞溅水花”）。
*   **机位随动**：镜头跟随速度变化（如“低机位快速跟摇”）。
*   *深入学习*：[👉 连续分镜与物理惯性推演心法](references/shot_logic_continuity.md)。

## 🎨 第二板块：妙笔 (Miaobi) 中文生图描述系统 (详细介绍)

本板块专为 **Midjourney、Stable Diffusion、Flux** 设计，统一使用“纯中文自然语言描述流”，消灭 AI 生图的“塑料感、平淡光影、千篇一律的构图”。

### 1. 正向中文生图六元公式
妙笔系统将中文生图的生成概率最大化，推演出了**「生图六元公式」**：
$$\text{通用中文提示词} = \text{[主体/妆造描述]} + \text{[环境与连锁互动]} + \text{[镜头/视线/构图]} + \text{[光影/色温/介质]} + \text{[微观肌理]} + \text{[限制性降噪句式]}$$

*   **主体妆造公式**：
    $$\text{完美角色提示词} = \text{角色身份} + \text{服装材质} + \text{发型发饰} + \text{面部妆容} + \text{色彩气质} + \text{当前状态}$$
*   *深入学习*：[👉 角色妆造与身份美学心法](miaobi-prompt-skill/references/character_styling_aesthetics.md)。

### 2. 限制性“剪词降噪”法则 (SD & Flux 负向成组)
我们严选出三组限制性“剪词降噪组”，彻底斩断 AI 生成时产生的粗糙细节与假高光：
```lic
(dirty details, muddy textures, dust particles:1.15), (plastic shine, waxy texture, fake specular highlights, oily skin:1.2), (halo glow, bloom effect, floating lights, misty haze:1.1), noisy background, low quality
```
*   *深入学习*：[👉 剪词降噪与减法美学心法](miaobi-prompt-skill/references/jianci_noise_reduction.md)。

## 🔄 第三板块：逆向拉片与灵魂还原心法 (详细介绍)

当用户发来参考图时，本系统将启动**「灵魂反推公式」**，锁定结构骨架三维，完美复刻：
$$\text{中文还原提示词} = \text{核心不变项（构图/色彩/氛围）} + \text{结构骨架（机位/光影/空间）} + \text{细节材质修饰} + \text{限制性降噪句式}$$
*   **微观肌理反推**：从拉丝金属、冰裂纹宣纸、次表面散射（SSS材质）及磨砂半透玻璃反推材质权重。
*   **物理对抗反推**：逆向还原空气流速感、重力阻尼感与体积雾光路。
*   *深入学习*：[👉 画面逆向反推与灵魂重构心法](miaobi-prompt-skill/references/reverse_engineering_soul.md)。
